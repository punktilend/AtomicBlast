# AtomicBlast

Self-hosted music streaming system. Torrented music flows from qBittorrent into Backblaze B2, then gets streamed via a Node.js proxy with on-the-fly transcoding. Dark green PWA theme (`#4ade80` accent).

## Repos

| Repo | Purpose |
|---|---|
| [AtomicBlast](https://github.com/punktilend/AtomicBlast) | Server + Web PWA (this repo) |
| [AtomicBlast-Win](https://github.com/punktilend/AtomicBlast-Win) | Windows desktop app |
| [AtomicBlast-Extension](https://github.com/punktilend/AtomicBlast-Extension) | Browser extension |

## Atomic Export Import

The Android app can now open a playlist JSON generated by the `Atomic Export` project from the Settings screen.

- Pick a playlist export JSON file from device storage
- The app matches imported tracks against the existing Backblaze B2 music library
- Matched tracks can be previewed and played immediately
- Unmatched tracks are listed so matching rules can be improved later
